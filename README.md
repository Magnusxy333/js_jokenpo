# 🎮 Jogo de Pedra, Papel e Tesoura (Jokenpô)

Aplicação web interativa desenvolvida em **JavaScript puro (Vanilla JS)**, **HTML5** e **CSS3**, com recursos modernos como persistência de dados local e modo de jogo automático (*Auto Play*).

## 🕹️ Como Executar o Projeto

Acesse esse link: https://magnusxy333.github.io/Jogo-de-Pedra-Papel-e-Tesoura-em-htlm-css-js/

1. Clone este repositório ou baixe os arquivos em sua máquina.
2. Certifique-se de manter a pasta `images/` com os ícones correspondentes.
3. Abra o arquivo `index.html` diretamente em qualquer navegador web, ou utilize a extensão **Live Server** (do VS Code) para rodar o projeto localmente.
